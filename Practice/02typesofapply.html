We have a three type to apply css in html page

  1. External Css
  2. Internal Css
  3. Inline Css
01. External css - External CSS is written on a separate page and it has to be linked with the HTML page, we link external CSS after the head section of HTML

02. Internal Css - Inline CSS is written in html head section. we create a style tag in head section and write a css property in a style tag.

03. Inline css - Inline css is written in singal element.we create a style attribue in perticular element and write a css property.
